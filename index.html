Olá! Sou Claudinei de Lima, desenvolvedor

Olá, meu nome é Claudinei e iniciei meus estudos na área de tecnologia em 2019 quando ingressei na Universidade Federal de Pelotas (UFPEL) no curso de Ciência da Computação. Desde então, adquiri conhecimentos em C, Java, Flutter, HTML, CSS, JavaScript e Banco de Dados. A partir de abril de 2022, entrei no mercado de trabalho como estagiário na Mertins Tecnologias, atuando como desenvolvedor FullStack e mobile. Minhas responsabilidades incluem o desenvolvimento e manutenção de códigos Java, Java EE e Java JPA para o back-end, utilização do JSF (JavaServer Faces) como framework para o front-end, gerenciamento de Banco de Dados com PostgreSQL e desenvolvimento de aplicações mobile com Flutter. Atualmente, estou focando meu tempo em aprofundar conhecimentos no desenvolvimento front-end e mobile, mas estou sempre em busca de novos desafios e oportunidades para meu contínuo desenvolvimento profissional.
